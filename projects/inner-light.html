Inner Light

2023-2024
Masters Thesis
Sole Contributor
9 Months

Project Overview

Inner Light is an innovative embodied VR meditation experience that bridges the gap between neuroscience and virtual reality. This groundbreaking project uses real-time brain activity (EEG) monitoring to create a dynamic, responsive virtual environment that visualizes the user's mental state through an interactive avatar.

As the sole contributor, I designed, programmed, and researched every aspect of this project, from the initial concept to the final implementation. This 9-month journey represents the culmination of my Masters thesis work.

Technical Implementation

Unity (C#) EEG Signal Processing Emotiv Software VR Development Real-time Data Avatar Animation

Unity Development

Built the entire VR experience using Unity and C#, creating a responsive and immersive environment that reacts to user input and brain activity in real-time. The system includes custom shaders, particle effects, and dynamic lighting that respond to meditation states.

EEG Integration

Integrated Emotiv software for real-time EEG signal processing, implementing algorithms to interpret brain wave patterns and translate them into meaningful visual and interactive elements within the VR environment.

Avatar System

Developed a sophisticated avatar system that represents the user's virtual body, responding to brain activity patterns with dynamic animations, color changes, and geometric transformations that reflect the user's mental state.

Research & Innovation

Key Research Areas

  • Neuroscience and VR integration
  • Real-time biofeedback systems
  • Embodied cognition in virtual environments
  • Meditation and mindfulness technology
  • Human-computer interaction through brain signals

This project represents a novel approach to combining neuroscience research with immersive technology. The real-time EEG integration allows users to see their mental state visualized in a way that has never been done before, creating a unique form of biofeedback that enhances meditation practice.

Technical Challenges

One of the most significant challenges was achieving real-time processing of EEG signals with minimal latency. This required optimizing both the signal processing algorithms and the Unity rendering pipeline to ensure smooth, responsive interactions.

Another major challenge was creating an intuitive and meaningful visualization system that could translate complex brain wave patterns into understandable visual feedback that enhanced rather than distracted from the meditation experience.

Impact & Future Work

This project demonstrates the potential for VR technology to serve as a bridge between our internal mental states and external digital environments. The implications extend beyond meditation to potential applications in therapy, education, and human-computer interaction.

Future work could explore expanding the system to include other biometric sensors, creating more sophisticated visualization algorithms, and developing applications for clinical and therapeutic use.
